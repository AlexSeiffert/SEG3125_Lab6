# SEG3125_Lab6

## How to Run This Project

1. **Install dependencies** (only needed the first time):

   ```sh
   npm install
   ```

2. **Start the server:**

   ```sh
   node scripts/app.js
   ```

3. **Open your browser and visit:**

   [http://localhost:3000/survey](http://localhost:3000/survey) — for the survey form

   [http://localhost:3000/results](http://localhost:3000/results) — for the results view

---

### Project Structure

- `scripts/app.js` — Main entry point, starts the Express server
- `scripts/server.js` — Route and server logic
- `views/` — Contains EJS and HTML views
- `public/` — Static assets (CSS, images, etc.)

---

**Dependencies:**

- express
- ejs

Install with `npm install` as shown above.
